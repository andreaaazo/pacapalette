# Color Palette Generator

This project features:
- OKLCH color space for better color manipulation
- Contrast ratio calculation with APCA method
- Color palette generation based on user input

## How it Works
1. The user inputs a base color in hex format.
2. The application converts the color to OKLCH color space.
3. The application generates a palette of colors based on the base color.
4. The application calculates the contrast ratio of each color in the palette using the APCA method and displays the results.

## Usage
1. Clone the repository
2. Install dependencies
3. Run the application
4. Input a base color and generate a Palette
